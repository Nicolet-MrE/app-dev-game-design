WebDev

Welcome!

In this project-based course, students will develop web applications and games with responsive design for use on multiple devices. Students are introduced to the fundamentals of app/game design and development in a VSCode development environment utilizing object oriented programming principles. Gaining a command of HTML, CSS, and Javascript, students will create mobile-first, interactive designs for fully immersive user experiences.

1.0 Ready, Get Set... | UX, VSCode & Git

This introductory unit establishes the technical and conceptual foundation for the course by focusing on the professional development environment and the principles of user-centered design. Students will explore the definition of User Experience (UX) and how game design principles can be applied to create more engaging digital products. Practically, students will set up their workspace using Visual Studio Code and learn to use GitHub for version control and collaboration, which are essential skills for modern software development.

  • Unit Details

2.0 The Digital Anatomy | HTML5 & the DOM

Students will transition from the workspace configuration of Unit 1 to the technical execution of web architecture. They will learn to build the "Skeleton" of a webpage using HTML5. The unit covers the hierarchical nature of the Document Object Model (DOM), the selection of semantic vs. non-semantic tags, and the use of attributes and inline properties to prepare elements for future styling and interactivity.

  • Unit Details

3.0 Skin with Style | CSS3

Students transition from structural markup to visual presentation. The unit covers the three ways to apply CSS (inline, internal, and external) and emphasizes the "Separation of Concerns" by focusing on external stylesheets. Students will master the CSS Box Model, learn the most frequently used properties for typography and color, and implement modern workflow techniques including native CSS nesting and CSS Variables for site-wide consistency.
